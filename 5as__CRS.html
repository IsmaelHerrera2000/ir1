1 Introducción

1.1 Alcance del proyecto

5as es una empresa dedicada a administrar negocios y fincas. Gestiona principalmente temas fiscales y fincas. Sus principales clientes serían autónomos y micro PYMEs.
Actualmente, utilizan libretas, documentos manuales y Excel para realizar sus principales tareas, pero no es nada óptimo por lo que sería muy útil usar un sistema de información actual y adaptado.

Se pretende diseñar un nuevo sistema de información que cumpla los siguientes puntos:

    -Software capaz de gestionar fincas, herencias y divorcios.   

    -Control de la contabilidad.   

    -Control de incidencias.   

    -Sistema de notificaciones.

1.3 Objetivos del proyecto

OBJ-0001 Proyecto 5as
Versión 1.0 (09/10/2022)
Autores
  • Herrera Ramírez, Ismael
Descripción

El sistema deberá actualizar la forma de trabajo de la empresa 5as, así como automatizar y facilitar sus tareas.
Subobjetivos
  • [OBJ-0005] Gestión de fincas, herencias y divorcios: El sistema deberá ser capaz de gestionar todas las tareas relacionadas con fincas, herencias y divorcios. Contará con una interfaz de usuario lo más fácil de usar posible.
  • [OBJ-0006] Control de la contabilidad: El sistema deberá facilitar los datos sobre la contabilidad así como realizar un balance contable.
  • [OBJ-0007] Control de incidencias: El sistema deberá permitir gestionar las incidencias de las fincas.
  • [OBJ-0008] Sistema de notificaciones: El sistema deberá de incluir un sistema de notificaciones que dependido del estado de una tarea, avise de acuerdo a los plazos de entrega.
Importancia vital
